BPM两步上手

只需两步：

1. <script src="http://bpmjs.org/api/combo??jquery"></script>
2. hui.require(['jquery'], function () { alert($); }

示例代码：（运行代码）

<!DOCTYPE html>
<html lang="en">
<head>
<meta charset="UTF-8">
<title>Demo - jquery</title>
<script src="http://bpmjs.org/api/combo??jquery"></script>
</head>
<body>
<div id="a">a</div>

<script type="text/javascript">
    hui.require(['jquery'], function () {
        // init
        document.getElementById('a').innerHTML = 'jquery ok';
        alert($('#a').innerHTML);
    });
</script>
</body>
</html>
